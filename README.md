# Instapro

MVP социальной сети для обмена фотографиями
В данном пректе реализовано:
- Интеграция верстки списка постов с API;
- Страница добавления поста;
- Страница с постами отдельного пользователя;
- функция лайков;
- изменено использование переменной post;
- исправлена валидация формы авторизации;
- исправлена валидация формы при регистрации, все сообщения об ошибке выводятся одним alert.
